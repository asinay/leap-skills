# Leap Skills — Quick Reference

## Start here (always, in this order)

| Step | Skill | Why |
|------|-------|-----|
| 1 | `/voice-builder` | Captures how you write and speak. Every other skill reads this. |
| 2 | `/audience-builder` | Defines who you're writing for. Content skills use this too. |

---

## Then use any skill below

### Executive Presence
| Skill | Use it when... |
|-------|---------------|
| `/executive-story` | You need career stories for interviews, keynotes, or LinkedIn |
| `/room-presence` | You have a high-stakes meeting, negotiation, or difficult conversation coming up |
| `/thought-leader-post` | You want to write a LinkedIn post from a real opinion or experience |

### LinkedIn & Personal Brand
| Skill | Use it when... |
|-------|---------------|
| `/profile-rewrite` | Your LinkedIn profile reads like a resume instead of a magnet |
| `/content-engine` | You don't know what to post — generates 30 ideas in one session |
| `/hook-generator` | You have a post topic but need a stronger opening line |

### Career Clarity
| Skill | Use it when... |
|-------|---------------|
| `/positioning-builder` | You struggle to explain what you do or why you're different |
| `/pivot-mapper` | You're considering a career change and need to map what transfers |
| `/next-move` | You're at a crossroads and need clarity before you start executing |

---

## Dependencies at a glance

- `/voice-builder` → required before everything else
- `/audience-builder` → required before content and brand skills
- `/content-engine` + `/thought-leader-post` + `/profile-rewrite` → read both foundation profiles
- `/hook-generator` + `/executive-story` + `/room-presence` + `/pivot-mapper` + `/next-move` → read voice profile only

---

## Suggested flows

**New user:** `/voice-builder` → `/audience-builder` → `/positioning-builder`

**Job search:** `/voice-builder` → `/executive-story` → `/positioning-builder` → `/pivot-mapper`

**LinkedIn growth:** `/voice-builder` → `/audience-builder` → `/profile-rewrite` → `/content-engine` → `/thought-leader-post`

**Big meeting:** `/voice-builder` → `/room-presence`
